Makoto Z BOT

Makoto Z, Ou apenas Makoto, E uma (Nova) bot discord multi modular de raça lobo. A sua principal função e entregar uma experiência musical para quem a usa, alem de outras categorias úteis.

Procurando um Comando?

Abaixo você tera a lista completa de todos os meus comandos disponíveis! Alem de exemplos e algumas descrições.

Utilidade

Comando Descrição Como Usar
ping Normalmente usado para verificar a latência, o ping E um comando público para ver se estou online. z/ping
help Usado claramente para ver a lista de comandos. z/help
avatar Use isso quando quiser roubar aquela foto do seu amigo. z/avatar, z/avatar @pessoa
emoji Um comando util pra quando você quiser talvez ver aquele emoji MAIOR! z/emoji *emojiname*
vote Veja a lista das Bot Lists, Que estou participando e ajude-me votando em mim. z/vote
update O comando update, tem a utilidade para o membro ver quais ultimos arquivo foram alterados no meu repositório principal. z/update
botinfo botinfo te servirá como um comando super util para visualizar diversas informações sobre mim. z/botinfo

Música

Comando Descrição Como Usar
play Use play para tocar músicas via, Nome Ou Link. z/play *Nome*, z/play *Link*
pause Pause a música por um tempo indeterminado até a repetição do mesmo. z/pause
stop Pare a música e junto me disconectarei do canal de voz atual. z/stop
autoplay Ao ativar, músicas semelhantes serão tocadas e/ou aleatórias do mesmo gênero. z/autoplay
volume Altere o volume da música tocando atual. z/volume *quantidade*
join Faça-me entrar no canal de voz que você está conectado. z/join
queue Veja a lista de reprodução e as próximas músicas a seguir. z/queue
loop Ative a repetição da música atual ou de toda lista de reprodução. z/loop (1 = música), z/loop(2 = Queue)
nowplaying Vejo o tempo da faixa da música tocando atualmente. z/nowplaying, z/np
status Veja uma lista de informações sobre a música e outros. z/status
avancar Avance a música atual em segundos. z/avancar *numero*
regressar Retorne a música tocando atualmente em alguns segundos z/regressar *número*
clearqueue Limpe todas as músicas da lista de reprodução, exceto a atual. z/clearqueue
playlist Veja uma lista de músicas por categoria já pré-feita. z/playlist, z/playlist *número*
radio Escute mais de 130 estações disponíveis da Makoto FM! z/radio, z/radio *estação*
removetrack Remova uma música específica da lista de reprodução. z/removetrack *número*
search Procure uma música específica no youtube e selecione a usando números. z/search *Nome*

Permissões

Algumas permissões da Makoto podem ser estranhas ou desconhecidas, Mais são SUPER! Necessárias para que ela funcione do melhor jeito possivel.

Abaixo você poderá ver algumas delas e junto descrições do, Porque diabos a Makoto usaria essa permissão!?

Permissões Necessárias
Ler Mensagens(Necessário para eu poder ler as mensagens) / Gerenciar Mensagens(Necessário para que eu consiga editar as Embed Pages) / Anexar Arquivos(Necessário para eu usar imagens do meu Canvas) / Adicionar Reações(Extremamente necessário para o sistema de música e outros) / Conectar(Necessário para me conectar no canal) / Falar(Necessário para poder tocar Músicas/Radios). (Necessário para usar meus emotes customizados)
Permissões Não Necessárias
Expulsar(Necessário para expulsar pessoas) / Banir(Necessário para banir pessoas) / Criar convites(Necessário para suporte).
